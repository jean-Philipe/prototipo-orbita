# Prototipo Orbita

Prototipo de una página de oportunidades para estudiantes, inspirado en el diseño de Airbnb.

## Estructura del Proyecto

- `index.html` - Página principal con el listado de oportunidades
- `oportunidad.html` - Página individual de oportunidad (pendiente de implementar)
- `styles.css` - Estilos CSS separados
- `script.js` - Lógica JavaScript para filtros y visualización
- `oportunidades.json` - Datos de las oportunidades en formato JSON
- `Imagenes/` - Carpeta con las imágenes de las oportunidades

## Características

- Diseño responsive inspirado en Airbnb
- Barra de búsqueda con filtros por:
  - Tipo de oportunidad
  - Fechas de postulación
  - Nivel educativo
- Cards de oportunidades con información relevante
- Sistema de filtrado dinámico
- Modales para selección de filtros

## Cómo usar

1. Abre `index.html` en un navegador web moderno
2. Los datos se cargan desde `oportunidades.json`
3. Usa los filtros en la barra de búsqueda para filtrar oportunidades
4. Haz clic en una oportunidad para ver más detalles (página pendiente)

## Para GitHub Pages

Este proyecto está listo para ser hosteado en GitHub Pages. Simplemente:
1. Sube todos los archivos al repositorio
2. Activa GitHub Pages en la configuración del repositorio
3. Selecciona la rama y carpeta raíz como fuente
